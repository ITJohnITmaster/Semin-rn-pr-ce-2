Notebook

Notebooky jsou počítače, které mají již zabudovanou klávesnici a monitor. Dále Notebook obsahuje základní myš (touchpad). Notebooky můžeme rozdělit podle OS - Windows, Linux, iOS.

Mezi Notebooky patří i Chromebooky s OS ChromeOS. Společnost Microsoft spravuje nejobvyklejší OS - Windows. iOS spravuje Apple a ChromeOS google. Linux není spravován jednou konkrétní společností.

Existují kancelářské notebooky (Numerická klávesnice, procesor přizbůsobený pro kanceláře), Herní notebooky (Procesor pro hry, špičková grafická karta, velké úložiště), ale také obyčejné domácí Notebooky.

Windows notebooky mají většinou předinstalované MS Aplikace (Teams, OneDrive, Excel, PowerPoint a další). MacBooky (iOS) jdou velmi jednoduše spárovat s mobilním zařízením a se vším Apple příslušenstvím
